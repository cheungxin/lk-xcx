## 使用流程

### 用户登录流程

1. 用户点击"我的"页面的头像或"点击登录"区域
2. 触发 `handleLogin()` 方法
3. 尝试静默登录（如果有 wxlogin 模块）
4. 如果需要授权，跳转到登录页面
5. 用户点击"手机号快捷登录"按钮
6. 微信弹出授权确认框
7. 用户同意授权后，获取手机号加密数据
8. 调用后端接口完成登录（目前是模拟数据）
9. 保存登录状态和用户信息
10. 返回到来源页面或首页

### 登录状态检查

在需要登录的功能中，添加以下代码：

```javascript
const isLogin = wx.getStorageSync('isLogin')
if (!isLogin) {
  tool.toast('请先登录')
  setTimeout(() => {
    this.handleLogin() // 或直接跳转到登录页
  }, 1500)
  return
}
```

## 存储的数据

登录成功后，会在本地存储以下数据：

- `isLogin`: true（登录状态标识）
- `token`: 访问令牌
- `userInfo`: 用户信息对象
  - `openId`: 微信 openId
  - `phone`: 手机号
  - `nickname`: 昵称
  - `avatarUrl`: 头像地址

## 后端接口对接

目前登录功能使用的是模拟数据，需要对接真实的后端接口：

### 需要的接口

**接口地址：** `/api/auth/phone-login`（示例）

**请求方法：** POST

**请求参数：**
```json
{
  "code": "微信登录凭证",
  "encryptedData": "手机号加密数据",
  "iv": "加密算法初始向量"
}
```

**返回数据：**
```json
{
  "code": 200,
  "message": "登录成功",
  "data": {
    "token": "访问令牌",
    "userInfo": {
      "openId": "用户openId",
      "phone": "手机号",
      "nickname": "昵称",
      "avatarUrl": "头像地址"
    }
  }
}
```

## 注意事项

1. **微信开放平台配置**
   - 需要在微信公众平台配置服务器域名
   - 需要开通"手机号快速验证组件"权限

2. **隐私协议**
   - 需要在小程序管理后台配置隐私协议
   - 首次使用需要用户同意隐私协议

3. **wxlogin 模块**
   - 如果项目中没有 wxlogin 模块，会直接跳转到登录页面
   - 可以根据需要实现 wxlogin 模块的静默登录功能

4. **登录状态管理**
   - 建议在 app.js 中添加全局登录状态检查
   - 可以在 onLaunch 时检查 token 是否过期
