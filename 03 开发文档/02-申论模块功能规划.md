## 总体架构说明

申论模块与行测共用同一套 UI 框架，但题目类型不同——申论题型为主观题（简答/作文），无法像行测单选题那样即时判对错。因此申论模块的数据存储和评分逻辑与行测有所区别。

申论模块在首页通过**科目 Tab 切换**（`selectedSubjectId: 'shenlun'`）进入，进入后专项练习面板和考场模式面板均切换为申论对应的知识点树和题目。

### 申论五大题型

| 题型 id | 题型名称 | 描述 |
|--------|---------|------|
| `summary` | 概括归纳 | 从材料中提炼要点 |
| `countermeasure` | 提出对策 | 针对问题提出解决方案 |
| `analysis` | 综合分析 | 多角度分析材料观点 |
| `execution` | 贯彻执行 | 公文写作、通知、报告等 |
| `essay` | 申发论述 | 申论大作文 |

## 模块一：申论专项练习

### 1.1 功能描述

用户切换到申论 Tab，在专项练习面板选择题型（如「概括归纳」），配置题数后进入申论答题页。申论答题页与行测答题页**分开**，使用专用的申论答题页面，支持主观作答（文本输入）。

### 1.2 页面流程

```
首页 → 切换 Tab 到「申论」
  → 专项练习面板显示申论五大类考点树
      → 点击考点（如「概括归纳」）→ 弹出练习设置面板
          → 点击「开始练习」
              → 申论答题页 /subpackages/training/shenlun/shenlun?type=summary&count=3
                  → 展示材料 + 问题
                  → 用户文本区域作答
                  → 交卷 → 参考答案展示页
```

### 1.4 shenlun.js 核心函数

```javascript
onLoad(options) {
  const { practiceType, type, count, date } = options || {}
  const questionType = type || 'summary'
  const questionCount = parseInt(count) || 3

  const questions = this.loadShenlunQuestions(questionType, questionCount, date)
  const answers = questions.map(() => '')
  const wordCounts = questions.map(() => 0)

  this.setData({
    practiceType: practiceType || 'practice',
    questionType,
    questions,
    answers,
    wordCounts,
    totalQuestions: questions.length,
    currentIndex: 0,
    currentQuestion: questions[0] || null,
    currentAnswer: '',
    currentWordCount: 0,
    startTime: Date.now(),
  })
  this.startTimer()
},

// 加载题目（从 mock/shenlun.js）
loadShenlunQuestions(type, count, date) {
  if (date) {
    return getDailyShenlunQuestions(date)
  }
  return getShenlunQuestions(type, count)
},

// 文本输入
onAnswerInput(e) {
  const val = e.detail.value
  const count = val.length
  const answers = [...this.data.answers]
  const wordCounts = [...this.data.wordCounts]
  answers[this.data.currentIndex] = val
  wordCounts[this.data.currentIndex] = count
  this.setData({
    answers,
    wordCounts,
    currentAnswer: val,
    currentWordCount: count,
  })
},

// 切题（保存当前作答）
switchQuestion(newIndex) {
  if (newIndex < 0 || newIndex >= this.data.totalQuestions) return
  const newQuestion = this.data.questions[newIndex]
  this.setData({
    currentIndex: newIndex,
    currentQuestion: newQuestion,
    currentAnswer: this.data.answers[newIndex],
    currentWordCount: this.data.wordCounts[newIndex],
  })
},

// 交卷
handleFinish() {
  const elapsed = Math.floor((Date.now() - this.data.startTime) / 1000)
  const result = {
    practiceType: this.data.practiceType,
    questionType: this.data.questionType,
    questions: this.data.questions,
    answers: this.data.answers,
    wordCounts: this.data.wordCounts,
    duration: elapsed,
    finishedAt: new Date().toISOString(),
  }
  storage.setShenlunResult(result)
  storage.appendRecord({
    type: '专项练习',
    title: `申论-${QUESTION_TYPE_NAMES[this.data.questionType] || '专项'}`,
    questions: this.data.totalQuestions,
    correct: 0,       // 主观题暂不自动评分
    accuracy: 0,
    duration: `${Math.floor(elapsed / 60)}分钟`,
  })
  this.setData({ showResult: true })
},
```

### 1.5 修改首页 `onTreeNodeTap` 支持申论

首页 `onTreeNodeTap` 中根据当前 `selectedSubjectId` 判断科目，申论科目跳转到申论答题页：

```javascript
onTreeNodeTap(e) {
  const { node } = e.detail
  if (!node || node.hasChildren) {
    tool.toast('请选择具体知识点')
    return
  }

  const { selectedSubjectId } = this.data

  if (selectedSubjectId === 'shenlun') {
    // 申论 → 直接弹出练习设置并标记为申论练习
    this.setData({
      showPracticeSettings: true,
      selectedFocusNode: node,
      isShenlunMode: true,
    })
  } else {
    // 行测 → 原有逻辑
    this.setData({
      showPracticeSettings: true,
      selectedFocusNode: node,
      isShenlunMode: false,
    })
  }
},

// 确认开始练习（兼容行测+申论）
handleConfirmPractice() {
  const { selectedSubjectId, practiceSettings, selectedFocusNode, isShenlunMode } = this.data
  if (!selectedFocusNode) return
  this.setData({ showPracticeSettings: false })

  if (isShenlunMode) {
    router.push('SHENLUN', {
      practiceType: 'practice',
      type: selectedFocusNode.id,
      count: practiceSettings.questionCount || 3,
    })
  } else {
    router.push('PRACTICE', {
      practiceType: 'practice',
      subject: selectedSubjectId,
      focus: selectedFocusNode.id,
      focusName: selectedFocusNode.name,
      count: practiceSettings.questionCount,
      year: practiceSettings.year,
    })
  }
},
```

### 6.1 功能描述

  const records = wx.getStorageSync('daily_records') || {}
  records[`shenlun_${date}`] = { ...result, done: true, subject: 'shenlun' }
  wx.setStorageSync('daily_records', records)
},

## 申论与行测的核心差异总结

| 对比维度 | 行测 | 申论 |
|---------|------|------|
| 题型 | 单选/多选（客观题） | 主观题（简答/作文） |
| 作答方式 | 点击选项 | 文本输入 |
| 即时反馈 | 选即判对错 | 提交后看参考答案 |
| 评分方式 | 自动计算正确率 | 用户自评或无评分 |
| 解析页 | 逐题显示对错+解析 | 对照参考答案+评分点 |
| 记录存储 | 正确率 accuracy 字段有值 | accuracy = 0，不统计正确率 |
| 每日一练 | 5道单选题 | 1-2道主观题+材料 |
