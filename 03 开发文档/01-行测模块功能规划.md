## 总体架构说明

行测模块共有三种练习类型，全部复用同一个答题页面 `practice.js`，通过路由参数 `practiceType` 区分来源，交卷时按类型分流写入不同存储键。

| 练习类型 | practiceType 值 | 入口 | 交卷后去向 |
|---------|---------------|------|---------|
| 每日一练 | `daily` | 首页快捷入口 → 专属每日一练页 | 解析页 |
| 专项练习（自主刷题） | `practice` | 首页考点树点击 → 进入答题 | 解析页 |
| 考场模式（模拟刷题） | `exam` | 首页考场模式面板 → 进入考试 | 考试报告页 |

## 模块一：每日一练（日历打卡）

### 1.1 功能描述

系统每天根据日期自动生成 5 道题（行测科目，难度中等），用户完成当天练习后在日历上显示"已打卡"徽章。用户可回顾历史打卡记录，点击历史日期可查看当天题目（只读）。

### 1.2 页面流程

```
首页快捷入口「每日一练」
  → 每日一练主页 /subpackages/training/daily/daily
      → 展示日历打卡状态 + 今日题目预览
      → 点击「开始练习」按钮
          → 答题页 /subpackages/training/practice/practice?practiceType=daily&date=YYYY-MM-DD
              → 答完 5 题后点击「交卷」
                  → submitPractice() 写入 storage.setDailyRecord(date, result)
                  → 跳转解析页 /subpackages/training/analysis/analysis?practiceType=daily
```

```javascript
submitPractice() {
  const { practiceState, practiceType, practiceDate } = this.data
  const correctCount = practiceState.answerSheet.filter(item => item.status === 'correct').length
  const accuracy = ((correctCount / practiceState.totalQuestions) * 100).toFixed(1)
  const elapsed = Math.floor((Date.now() - this.data.startTime) / 1000)

  const result = {
    totalQuestions: practiceState.totalQuestions,
    correctCount,
    accuracy,
    answerSheet: practiceState.answerSheet,
    questions: practiceState.questions,
    duration: elapsed,
    finishedAt: new Date().toISOString(),
  }

  // 写入公用结果（供解析页读取）
  storage.setPracticeResult(result)

  // 按类型分流写入练习记录
  if (practiceType === 'daily') {
    storage.setDailyRecord(practiceDate, result)
    storage.appendRecord({
      type: '每日一练',
      title: `行测每日一练 ${practiceDate}`,
      questions: result.totalQuestions,
      correct: result.correctCount,
      accuracy: parseFloat(result.accuracy),
      duration: `${Math.floor(elapsed / 60)}分钟`,
    })
  } else if (practiceType === 'practice') {
    storage.appendRecord({
      type: '专项练习',
      title: practiceState.paperTitle || '专项练习',
      questions: result.totalQuestions,
      correct: result.correctCount,
      accuracy: parseFloat(result.accuracy),
      duration: `${Math.floor(elapsed / 60)}分钟`,
    })
  }

  router.push('ANALYSIS', { practiceType })
},
```

## 模块三：考场模式（模拟刷题）

### 3.1 功能描述

考场模式分三种子类型：
- **专项考试**：选择一个或多个知识点出题，有时间限制，完成后出考试报告
- **混合考试**：选择多个知识点混合出题，有时间限制，完成后出考试报告
- **全套考试**：选择一套完整试卷作答，有时间限制，完成后出考试报告

三种模式统一走 `exam.js` 页面，交卷后跳转**考试报告页**（新建），报告页可点击进入全卷解析页。

### 3.2 页面流程

```
首页考场模式面板
  → 选择模式：专项/混合/全套
      → 专项/混合：选择知识点 + 配置出题设置（时间/题数）
      → 全套：选择试卷
  → 点击「开始考试」按钮（examReady=true 时激活）
      → 答题页 exam?examMode=special&subject=xingce&categoryIds=logic,verbal&duration=20&count=30
          → 倒计时 + 答题
          → 交卷（手动/超时自动）
              → submitExam() 生成报告数据
              → 写入 storage.appendRecord(type='模拟刷题')
              → 跳转考试报告页 /subpackages/training/exam-report/exam-report
                  → 查看得分/各板块正确率
                  → 点击「全卷解析」→ analysis?practiceType=exam
                  → 点击「错题解析」→ analysis?practiceType=exam&filter=wrong
```

```javascript
    finishedAt: new Date().toISOString(),
  }

  // 写入存储
  storage.setExamResult(examResult)
  storage.appendRecord({
    type: '模拟刷题',
    title: currentPaper.name,
    questions: questions.length,
    correct: correctCount,
    accuracy: Math.round((correctCount / questions.length) * 100),
    duration: `${Math.floor(elapsed / 60)}分钟`,
  })

  // 跳转报告页（替换原来的 showResult 内联展示）
  router.push('EXAM_REPORT')
},
```

```javascript
onLoad(options) {
  const { practiceType, filter } = options || {}
  
  let result = null
  let pageTitle = '全卷解析'
  
  if (practiceType === 'exam') {
    result = storage.getExamResult()
    pageTitle = filter === 'wrong' ? '错题解析' : '全卷解析'
  } else {
    result = storage.getPracticeResult()
    pageTitle = practiceType === 'daily' ? '今日解析' : '全卷解析'
  }
  
  if (!result) {
    tool.toast('暂无解析数据')
    return
  }
  
  let questions = result.questions || []
  
  // 应用筛选
  if (filter === 'wrong') {
    const wrongIds = (result.answerSheet || [])
      .filter(a => a.status === 'wrong')
      .map(a => a.id)
    questions = questions.filter(q => wrongIds.includes(q.id))
  }
  
  this.setData({ 
    pageTitle, 
    questions, 
    answerSheet: result.answerSheet || [],
    filter: filter || 'all',
  })
},
```

### 4.2 storage 键名规划

| 键名 | 存储内容 | 使用场景 |
|-----|---------|---------|
| `practice_cache` | 当前答题中间状态 | 答题卡跳转/刷新恢复 |
| `practice_result` | 最近一次专项练习结果 | 解析页读取 |
| `exam_result` | 最近一次考场模式结果 | 考试报告页 + 解析页读取 |
| `daily_records` | 全部每日一练历史 `{date: result}` | 日历打卡页 |
| `practice_records` | 全部练习记录列表（最多200条） | 记录页读取 |
