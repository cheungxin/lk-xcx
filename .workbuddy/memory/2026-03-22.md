## TabBar 页面切换修复

### 问题诊断
TabBar 点击无法切换页面，原因如下：
1. tabbar 组件使用 `wx.switchTab` 跳转，但 3 个 tab 页面在分包中，不在 `app.json` 的 `tabBar.list` 里，导致跳转全部失败
2. `profile.wxml` 引用了不存在的 `custom-tabbar` 组件（应该是 `tabbar`）
3. 分包页面（exam/quick/review）没有引入 tabbar 组件
4. 存在两套 tabbar 实现（`custom-tab-bar/` 和 `components/tabbar/`），互相冲突

### 修复方案
1. **tabbar.js**：将 `wx.switchTab` 改为 `wx.reLaunch`，兼容主包和分包页面跳转
2. **profile.wxml/json**：修正组件名 `custom-tabbar` → `tabbar`，并在 json 中注册
3. **exam/quick/review**：三个分包页的 wxml 和 json 都添加了 tabbar 组件（selected 分别为 1/2/3）
4. **app.json**（两个）：tabBar.list 移除了 login 页，只保留首页和我的；清理了无用的 custom-tab-bar 引用
5. **删除并重建** `miniprogram/custom-tab-bar/` 目录——作为微信 custom tabBar 机制的代理，内部引用 `components/tabbar/tabbar`
6. **修复 tabbar.wxss** 标签选择器问题——移除了 `t-icon` 标签选择器（组件 wxss 不允许）
