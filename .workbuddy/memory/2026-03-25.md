## 修复 WXML 中的 JavaScript 方法调用错误

### 问题
WXML 编译错误：`Bad value with message: unexpected token '.'`

### 原因
WXML 模板中直接调用了 JavaScript 对象方法（如 `.toFixed(1)`），但小程序 WXML 不支持直接调用方法，只能使用简单表达式。

### 修复文件
1. `subpackages/course/list/list.wxml` - 添加 WXS 模块 `util.formatPlayCount()`
2. `subpackages/course/detail/detail.wxml` - 添加 WXS 模块 `util.formatCount()`

### 解决方案
使用 WXS（WeiXin Script）在 WXML 中定义格式化函数，然后在模板中调用 WXS 模块的方法。

## 修复 WXML 中的数组方法调用错误

### 问题
WXML 编译错误：`unexpected '>' at pos58`

### 原因
WXML 模板中直接使用了 JavaScript 数组方法（如 `.filter()`），但小程序 WXML 不支持调用数组/对象方法。

### 修复文件
1. `subpackages/training/exam/exam.wxml`
2. `subpackages/training/exam/exam.js`
