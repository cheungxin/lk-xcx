# 图标资源目录

本目录用于存放小程序的图标资源。

## 使用说明

1. 将图标文件放置在此目录下
2. 在页面中使用相对路径引用：`/miniprogram/assets/icons/xxx.png`
3. 建议使用 SVG 格式或 IconFont

## 图标优化建议

- 优先使用 IconFont（体积小，可缩放）
- SVG 格式适合复杂图标
- PNG 格式注意使用 2x/3x 适配不同屏幕
- 统一图标尺寸规范（如 24x24, 48x48）

## 常用图标

- `back.png` - 返回图标
- `home.png` - 首页图标
- `search.png` - 搜索图标
- `settings.png` - 设置图标
- `close.png` - 关闭图标
