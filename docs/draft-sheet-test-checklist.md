# 草稿纸组件测试清单

## 功能测试

### 基础功能
- [ ] 点击右下角草稿纸按钮，能正常打开全屏草稿纸
- [ ] 草稿纸以半透明遮罩形式覆盖整个屏幕
- [ ] 可以看到底下的题目内容
- [ ] 点击关闭按钮能正常关闭草稿纸

### 绘画功能
- [ ] 可以在屏幕上正常书写
- [ ] 画笔流畅，没有延迟
- [ ] 画笔粗细合适（3px）
- [ ] 浅色模式下画笔为黑色
- [ ] 深色模式下画笔为白色
- [ ] 单点触摸显示为圆点

### 工具栏功能
- [ ] 撤销按钮：能撤销最后一笔
- [ ] 撤销按钮：没有内容时为禁用状态
- [ ] 重做按钮：撤销后能重做
- [ ] 重做按钮：没有可重做内容时为禁用状态
- [ ] 清空按钮：弹出确认对话框
- [ ] 清空按钮：确认后清空所有内容
- [ ] 清空按钮：没有内容时为禁用状态

### 事件阻止
- [ ] 在草稿纸上书写时，不会触发底下的选项按钮
- [ ] 在草稿纸上滑动时，不会滚动底下的页面
- [ ] 在草稿纸上点击时，不会触发底下的任何交互

### 数据持久化
- [ ] 书写内容会自动保存
- [ ] 关闭草稿纸后重新打开，内容还在
- [ ] 切换到下一题，草稿纸内容清空（新的 sceneKey）
- [ ] 返回上一题，能看到之前的草稿内容
- [ ] 触发按钮上有橙色小圆点提示（有内容时）

### 主题切换
- [ ] 切换到深色模式，画笔变为白色
- [ ] 切换到深色模式，背景变为深色半透明
- [ ] 切换到浅色模式，画笔变为黑色
- [ ] 切换到浅色模式，背景变为浅色半透明
- [ ] 工具栏图标颜色随主题变化

### 占位提示
- [ ] 空白时显示"在屏幕上书写草稿"提示
- [ ] 开始书写后提示消失
- [ ] 清空后提示重新显示
